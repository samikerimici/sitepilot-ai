# Politique de sécurité

## Versions prises en charge

La version actuellement testée est `1.1.0-rc1`. Les versions antérieures ne sont pas maintenues dans ce dépôt d’apprentissage.

## Signaler une vulnérabilité

Merci de ne pas publier de détail exploitable dans une issue publique.

Envoyez un message à **contact@lorraine.digital** avec :

- la version concernée ;
- les étapes de reproduction ;
- l’impact potentiel ;
- toute preuve utile ;
- vos coordonnées si vous souhaitez être recontacté.

Un accusé de réception sera envoyé dès que possible. La correction et la communication publique dépendront de la gravité et de la reproductibilité du problème.

## Secrets

Ne joignez jamais de vraie clé API, de mot de passe, de sauvegarde client ou de donnée personnelle à une issue, une pull request ou un commit.
