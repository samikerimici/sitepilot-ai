# Studio de contenu

Le studio permet de préparer plusieurs types de contenus :

- actualité ;
- article ;
- page ;
- FAQ.

## Informations demandées

Selon l’écran, renseignez le sujet, l’objectif, la ville, le ton, les mots-clés et la longueur souhaitée.

## Résultat

Le plugin crée un brouillon WordPress avec :

- un titre ;
- un contenu HTML simple ;
- un extrait ;
- une méta-description Yoast SEO lorsqu’elle est disponible.

## Bonnes pratiques

1. Relisez le brouillon.
2. Vérifiez les affirmations, dates et informations commerciales.
3. Ajoutez vos exemples et éléments différenciants.
4. Ajoutez une image et ses crédits si nécessaire.
5. Contrôlez le rendu mobile et le référencement avant publication.
