# FAQ

## SitePilot AI publie-t-il automatiquement les contenus ?

Non. La version actuelle crée des brouillons à relire avant publication.

## Une clé OpenAI est-elle obligatoire ?

Non. Le mode local fonctionne sans clé et génère une structure de contenu.

## L’analyse remplace-t-elle un audit professionnel ?

Non. Elle fournit des indicateurs et priorités, mais ne remplace pas un audit SEO, sécurité, performance ou accessibilité complet.

## Puis-je l’installer directement sur un site client ?

Cette version est une release candidate. Testez-la d’abord sur une préproduction et effectuez une sauvegarde.

## Pourquoi certaines méta-descriptions sont-elles enregistrées pour Yoast ?

La version actuelle utilise principalement la clé `_yoast_wpseo_metadesc`. Une compatibilité plus large pourra être ajoutée ultérieurement.

## Où sont conservés les réglages ?

Dans les options de la base WordPress. Ne mettez jamais une exportation de base contenant une clé API dans le dépôt GitHub.
