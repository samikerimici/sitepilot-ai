# Analyse et rapports

L’analyse attribue des scores à plusieurs catégories :

- SEO ;
- contenu ;
- images ;
- sécurité ;
- mises à jour.

Le score global est une synthèse pondérée de ces indicateurs.

## Contrôles réalisés

La version actuelle examine notamment :

- les images sans texte ALT ;
- les contenus très courts ;
- les méta-descriptions manquantes ;
- les mises à jour WordPress et extensions détectées ;
- l’utilisation de HTTPS ;
- le nombre d’administrateurs ;
- la récence de la dernière publication.

## Limites

Cette analyse est indicative. Elle ne remplace pas :

- un audit SEO complet ;
- un scanner de vulnérabilités ;
- un audit d’accessibilité ;
- une analyse de performance avancée ;
- une revue humaine.

## Historique

Les 30 dernières analyses sont conservées pour suivre l’évolution du score et des catégories.
